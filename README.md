## Projeto de Análise - Registro Civil (IBGE)

##Objetivos do Projeto
- Criar banco de dados do Registro Civil.
- Inserir dados oficiais do IBGE.
- Realizar consultas de:
  - total de nascimentos  
  - total de óbitos  
  - total de casamentos  
  - total de divórcios  

##Tecnologias Utilizadas
- MySQL 8.0
- Python 3.10 (opcional para ETL e automações)
- Workbench (opcional para modelagem e consultas)
